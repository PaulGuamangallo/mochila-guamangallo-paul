Para conectarnos a traves del protocolo SSH es necesario prepara nuestra maquina virtual o servidor linux
	(es necesario conocer la dir ip de nuestra VM o server linux)
	(tener en cuenta el user y contraseña)
	intallar el paquete ssh (openssh-server)
	verificar q el servicio este activo
        *Por defecto el servicio ya esta activo, si no es necesario activarlo a traves de comandos adjuntados en la doc$- Pc remoto
        Es necesario habilitar el cliente telenet ya sea a graficamente o en linea de comandos cmd administrador

Una ves completada la preparacion; ponemos el siguiente comando en cmd:
        ssh usuario@server
		ejemplo: ssh root@192.168.1.1
        *posteriormente nos pedira un usuario y contraseña
Una vez verificado las credenciales ya estaremos dentro de nuestro server linux o VM y podremos acceder al shell de la $***
